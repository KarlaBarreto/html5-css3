Principais Formatações

Negrito/Destaque

Neste paragrafo, irei adicionar uma frase em negrito (DESTAQUE) usando a tag STRONG

Italico/Enfase

Nesta frase terei uma enfase em italico usando a TAG EM

Marca Texto/Destque

Usando nessa frase um marca texto para dar destaque em determinado ponto usando a TAG MARK

Também posso usar outras cores de marca texto

Texto Grande e Pequeno

Estou criando variação em uma frase com palavras grandes e palavras pequenas usando as TAGS big e small consecutivamente

Texto "deletado"

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado usando a TAG DEL

Texto "inserido"

Podemos marcar um texto como inserido para dar enfase,sublinhar e indicar que ele foi adicionado ao site depois usando a TAG INS

Texto Sobrescritos

Serve para inserir numeros elevados em formulas por exemplo: x2+3=? USANDO A TAG SUP

Texto Subscrito

Serve para inserir numeros rebaixados em elementos quimicos por exemplo: H2O USANDO A TAG SUB
