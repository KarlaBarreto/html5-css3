Listas com HTML5

Listas Ordenadas

Listas ordenadas servem para listagem de coisas importantes. 
ex: aprovação do vestibular, receitas etc.

Para a criação das listas ordenadas usamos as TAGS OL e LI

  1. Maria
  2. João
  3. Vitoria

Para ordenar essas listas usamos o parametro TYPE 
dentro da tag ol e escolhemos as opções de marcações.
Ficando assim:

  1. Maria
  2. João
  3. Vitoria

Listas Não Ordenadas

    As listas não ordenadas são listas 
    cotidianas onde a ordem dos itens 
    não influencia no significado

Para criar esse tipo de lista usamos a TAG UL e IL

  • Arroz
  • Sabão
  • Feijão

Nessas listas as marcações são diferentes, feitas por simbolos. Ficando assim:

  • Arroz
  • Sabão
  • Feijão
